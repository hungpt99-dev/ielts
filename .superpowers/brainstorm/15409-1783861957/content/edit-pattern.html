How should users edit the roadmap?

Pick the interaction pattern that feels best

A

✏️ Inline Editing

Click any text (objective, skill, name) → it becomes an editable input. Save on blur/enter. Clean and fast.

B

Sliding Edit Panel

Click "Edit" → a panel slides down within the card showing all editable fields. Modal-like but contextual.

C

Modal Dialog

Click "Edit" → a centered modal opens with form fields. Feels more like a traditional "edit" experience.

D

Dedicated Edit Mode

Toggle "Edit Mode" at top of page → entire roadmap becomes editable (reorder handles, add buttons). Best for large restructures.
