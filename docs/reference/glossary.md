# Glossary

| Term | Definition |
|---|---|
| **Learning objective** | The goal of a learning session — what skill or knowledge gap the session addresses. Contains type, description, success criteria, and priority. |
| **Learning session** | A cohesive study unit focused on one skill. Contains one or more activities/exercises. Has a lifecycle: prepared → in-progress → completed/abandoned. |
| **Activity** | A step within a learning session. Can be a content review, an exercise, or a reflection. Activities are ordered within a session. |
| **Exercise** | A set of questions on a specific topic or skill. Contains 1+ questions of various types (MC, gap-fill, essay, etc.). Has an evaluation policy that determines how answers are graded. |
| **Attempt** | A learner's submission of answers to an exercise. Status: started → in-progress → submitted → evaluated. Tracks answers, time spent, and hints used. |
| **Evaluation** | The result of grading an attempt. Deterministic for objective types (MC, TFNG), AI-assisted for subjective types (essay, speaking). Produces scores, feedback, and mistake/skill evidence. |
| **Outcome** | The aggregated result of completing an exercise within a session. Contains score, accuracy, estimated band, mistakes, strengths, and vocabulary evidence. |
| **Evidence** | A detected fact about the learner's performance. Categories: mistake evidence, skill evidence (strength/weakness/improvement/plateau), progress evidence (milestone/trend). |
| **Learner context** | A snapshot of the learner's current state used by the AI tutor. Includes profile, exam context, roadmap progress, skill states, mistake summary, vocabulary summary, activity summary, and preferences. |
| **Tutor memory** | Persistent knowledge the AI tutor keeps about the learner across sessions. Includes weak points, mistake patterns, preferences, goals, strategies, open learning loops, and recommendation history. |
| **Roadmap task** | A scheduled study item in the learner's study plan. Has a date, category, title, estimated duration, completion status, and optional accuracy score. |
| **Proactive intervention** | An unsolicited message from the AI tutor triggered by a learner state change. Examples: exam countdown warnings, streak celebrations, mistake pattern alerts. Has deduplication and cooldown policies. |
| **Skill module** | A pluggable component that generates skill-specific exercises and evaluations. Available modules: Reading, Writing, Listening, Speaking, Vocabulary, Grammar. Registered in `SkillRegistry`. |
| **Study plan** | A structured, multi-week IELTS preparation roadmap with phases, weekly objectives, and daily tasks. Generated by `DailyPlanEngine` based on the user's profile. |
| **Spaced repetition** | An SM-2-based algorithm for scheduling vocabulary review intervals. Ratings: again (reset), hard (slow), good (standard), easy (fast). Updates interval, ease factor, and repetition count. |
